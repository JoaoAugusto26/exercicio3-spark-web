# Exercício 3: Integração Spark e PostgreSQL

Este repositório contém a solução para a atividade prática de back-end da disciplina. O objetivo foi criar uma aplicação Web em Java utilizando o framework Spark, conectada a um banco de dados PostgreSQL.

## ⚙️ Funcionalidades
A aplicação expõe um servidor web que permite:
- **Exibir um formulário HTML** para cadastro de usuários.
- **Receber requisições POST** via web para inserir novos usuários no banco.
- **Listar os usuários** cadastrados diretamente na página web.

## 🛠️ Tecnologias Utilizadas
- Java 21 + Eclipse
- Spark Framework (Servidor Web Embutido)
- PostgreSQL (Banco de Dados) + pgAdmin
- Maven (Gerenciamento de Dependências no `pom.xml`)
- Padrão de Projeto DAO
